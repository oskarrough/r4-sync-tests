# R5

A prototype exploring the future of Radio4000. Local-first (kinda) music player with R4 as the remote server.

## Deploying

The `main` branch deploys to https://pg.radio4000.com for now.

## Documentation

See the [`docs`](./docs) folder.

## Developing

```bash
bun install
bun dev
```
